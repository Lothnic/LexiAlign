# LexiAlign

For non-nerds this is a word matching engine for any Language Pair without the need to train a model(using parallel data).

## Special Thanks to CIS, LMU
This is a spiritual implementation of SimAlign by Centre for Language and Information Processing, University of Munich
- Code - https://github.com/cisnlp/simalign
- Paper - https://arxiv.org/pdf/2004.08728
